# Guess-the-Number-game.
Guess the Number — bu juda oddiy va hamma yoshdagilar oʻynashi mumkin bo‘lgan mantiqiy o‘yin. Unda kompyuter 1 dan ma’lum bir oralig‘dagi (masalan, 1–100) tasodifiy son o‘ylaydi, sizning vazifangiz esa shu sonni topish.

O‘yinning jarayoni quyidagicha:

Kompyuter bir son o‘ylaydi (siz bu sonni ko‘rmaysiz).

Siz taxminingizni kiritasiz.

Kompyuter sizga “kattaroq ayting” yoki “kichikroq ayting” deb maslahat beradi.

Siz to‘g‘ri sonni topguningizcha davom etasiz.

Yakunda nechta urinishda topganingiz chiqariladi.

Bu o‘yin:

Asabiylashtirmaydi (timer yo‘q, shoshilish shart emas),

Mantiqiy fikrlashni rivojlantiradi,

Kichiklar va kattalar uchun teng darajada qiziqarli.
